# Sales Analysis Portfolio Project

## Project Description

This project analyzes sales data using Python and data analysis techniques.

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Jupyter Notebook

## Key Findings

* Total Sales: 10,032,628.85
* Best Product: Classic Cars
* Best Country: USA
* Best Month: November
* Best Customer: Euro Shopping Channel
